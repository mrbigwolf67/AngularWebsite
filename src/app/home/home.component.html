Läs mer

  • Angular är en plattform och ramverk för att bygga klientapplikationer i HTML och TypeScript.

  • React är ett JavaScript-bibliotek skapat av Facebook. React är ett verktyg för att bygga User-interface-komponenter.

  • Node är en JavaScript runtime-miljö och körs på olika plattformar (Windows, Mac OS, etc.) Node använder JavaScript på servern.

  • HTML5 är den senaste versionen av Hypertext Markup Language, koden som beskriver webbsidor.

  • CSS3 är utformad för att möjliggöra separering av presentation och innehåll, inklusive layout, färger och teckensnitt.

  • SASS är ett skriptspråk som tolkas eller kompileras till Cascading Style Sheets (CSS).

  • Firebase / Firestore är Googles mobilplattform som hjälper till att snabbt utveckla högkvalitativa appar och webbsidor.

  • WordPress är ett online-verktyg för att skapa webbsidor med öppen källkod skrivet i PHP.

  • Apputvecklingen går framåt, och nu kan man programmera i ett språk för både iOS och Android telefoner.

  • Jag utgår från principen Mobile first, som innebär att man först designar för mobil innan man designar för webb mm.

  • JavaScript är programmeringsspråk för webben. JavaScript kan uppdatera och ändra både HTML och CSS.

  • TypeScript är ett öppet programmeringsspråk. Den förenklar JavaScript-koden, vilket gör det lättare att läsa, felsöka och testa
